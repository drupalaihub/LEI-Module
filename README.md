## Getting Started

To get started with the LEI Drupal Test, follow these steps:

1. Clone this repository:
   
   ```
   git clone https://github.com/kaarele/lei-drupal-test.git
   ```

2. Install dependencies:
   
   ```
   composer install
   ```

3. Start the Drupal server:
   
   ```
   drupal server
   ```
